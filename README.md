# Crisis-in-Quick-Bites-Food-Startup-
Analyzed operational and customer-service challenges faced by a food startup, focusing on delayed deliveries, service disruptions, and customer dissatisfaction. The project highlights key risk factors and data-driven insights to improve efficiency and customer retention.
